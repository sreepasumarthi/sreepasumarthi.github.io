CSCE 242: Client-Server Computing

Assignments

Assignment 1 - Basic HTML

In this assignment, students learn the fundamental elements of HTML. These include headings, paragraphs, and more. Based on a topic of choice, students will create a simple web page utilizing these fundamentals.

Assignment 2 - Basic CSS

In this assignment, students learn the fundamental elements of CSS. This includes styling using color themes, padding, and more. Based on a topic of choice, students will create and style a simple web page utilizing these fundamentals.

Assignment 3 - Page Layout

In this assignment, students learn how to create a page layout. Utilizing features such as flexbox, students are tasked to create a web page based on a given template. Moreover, students learn how to modify large/small resolutions.

Assignment 5 - Recreate CSS Page

In this assignment, students learn how to recreate a real-life CSS page example. They are asked to create is the University of South Carolina's College of Engineering & Computing Page.

Assignment 7 - Introduction to JS

In this assignment students learn how to integrate HTML, CSS, and JavaScript to create an interactive web page. On this web page, users can change an image, rotate and image, or add stars in a horizontal line.

Assignment 8 - Conditionals

In this assignment, students utilize what they've learned in class about conditionals and is statements to create a webpage with two exercises showcasing their understanding of the concepts.

Assignment 9 - Loops

In this assignment, students are tasked with creating a web page that displays two exercises. The first exercise is a bouncing ball and the second exercise displays text to the right of an image when clicked.
