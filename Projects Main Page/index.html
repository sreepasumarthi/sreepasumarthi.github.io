Project

Part 1 - Topic Selection

In this assignment, students are tasked with selecting a topic for their semester-long project. The topic may be in the interest of a client or the student themselves. Here, details about pages & the website are included.

Part 2 - Wireframe

In this assignment, students learn how to use the Mockitt prototyping tool to create a wireframe for their project. This wireframe will allow students to visualize their web page before beginning to develop it.

Part 3 - HTML & CSS

In this assignment, students implement their project wireframe using HTML & CSS. The main focus here is layout over colors & design. It serves as rough draft.

Part 4 - Colors, Pictures, and Text

Students make their HTML & CSS rough draft to a final version of their design (without javascript implementation yet). Design should look nice here.

Part 4.5 - Improve Your Website

In this assignment, students make their HTML & CSS rough draft to a final version of their design (without javascript implementation yet). The design should look nice. This is an extension of part 4.

Part 5 - JSON File Parsing

In this part of the assignment, students learn how to imlement JSON file parsing to their website instead of simply hard coding things like specific lists that appear on the website. This is more efficient.
